### 当ブログ構築の裏話
昨夜、Hugo と GitHub を使って個人ブログを作ろうとし、Stack テーマを適用してみたのですが、何度やってもエラーが出てしまいました。短時間で終わると思っていた作業が、結局 4月24日から4月25日までの2日がかりになり、かなりストレスがたまりました。

最初は設定方法を紹介する動画ばかり見ていたものの、やはり動画だけでは細かい部分が理解しづらく、途中から文字ベースの解説を読むことにしました。そこで、Hugo と Stack の公式サイトにある構築ガイドを確認し始めました。

その中で、Stack の作者が stack-starter を強く勧めていたので、次のリンクから試してみました。
https://github.com/CaiJimmy/hugo-theme-stack-starter

実際に使ってみると、やはり非常に簡単で、わずか十数分で自動デプロイ可能なブログを構築できました。

やっぱり文字による解説は効率は高いです。
将来は、しっかり勉強とするとき、文章で書かれたガイドは最優先にすること覚悟しました。
